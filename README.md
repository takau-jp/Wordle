# Wordle

C++ で作る、ターミナル版 `Wordle` です。

5 文字の英単語を 6 回以内で当てるゲームを作ることを目標にしています。
このリポジトリでは、`Makefile` を使ってビルドできます。

## できること

現時点では、ターミナルで遊べる基本版まで入っています。

- `make` でビルドできる
- `./wordle` で起動できる
- 5 文字以外の入力を弾ける
- 辞書にない単語を弾ける
- 判定結果を色付きで表示できる
- `./wordle apple` のように固定単語でテストできる

## 必要なもの

- C++17 に対応した `c++`
- `make`

macOS や Linux の一般的なターミナル環境を想定しています。

## 動かし方

リポジトリのルートで、次の順に実行します。

```sh
make
./wordle
```

ランダムではなく、正解語を固定して試したいときは次のように実行します。

```sh
./wordle apple
```

この場合、正解語は `apple` になります。

ビルド済みファイルを消したいときは、次を使います。

```sh
make clean
```

実行ファイルも含めて全部消したいときは、次です。

```sh
make fclean
```

最初からビルドし直したいときは、次を使います。

```sh
make re
```

## ファイル構成

- `Makefile`: ビルド設定
- `data/words.txt`: ゲームで使う辞書
- `REQUIREMENTS.md`: 要件定義
- `DESIGN.md`: 設計メモ
- `include/`: ヘッダファイル
- `src/`: 実装ファイル

## 使用ライブラリ

このプロジェクトでは、外部ライブラリは使っていません。
C++ の標準ライブラリだけで実装しています。

- `<iostream>`: ターミナルへの表示と入力に使用
- `<string>`: 単語やメッセージの文字列処理に使用
- `<vector>`: 推測履歴や文字ごとの判定結果の保持に使用
- `<unordered_set>`: 辞書に単語が存在するかを高速に調べるために使用
- `<fstream>`: `data/words.txt` の読み込みに使用
- `<random>`: 正解語をランダムに選ぶために使用
- `<array>`: 文字の出現回数を数えて判定するために使用
- `<cctype>`: 英字判定や小文字化、大文字表示に使用
- `<stdexcept>`: 辞書読み込み失敗や入力異常時のエラー処理に使用
- `<exception>`: `main` 関数で例外をまとめて受けるために使用

## 現在の注意点

- 引数なしの `./wordle` は辞書からランダムに正解語を選びます
- `./wordle apple` のように 5 文字単語を渡すと、その単語を正解語として使えます
- 色表示は ANSI エスケープシーケンスを使っています
